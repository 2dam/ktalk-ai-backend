# K-talk 웹 버전

데스크톱·태블릿·모바일 브라우저용 반응형 웹사이트입니다.

K-콘텐츠 기반 AI 한국어 학습 서비스의 반응형 프런트엔드입니다.

## 실행

`index.html`을 브라우저에서 열거나 정적 파일 서버로 실행하세요.

- `index.html`: 서비스 홈페이지
- `signup.html`: 진단형 회원가입
- `dashboard.html`: 개인화 구독자 대시보드

## 포함된 기능

- K-드라마, K-POP, 여행, 음식 주제 선택
- 대화 메시지 입력 및 AI 튜터 응답 데모
- 오늘의 표현을 입력창으로 가져오기
- 모바일/태블릿/데스크톱 반응형 디자인
- `signup.html`: 학습 유형 진단과 맞춤 교재·학습 전략을 제공하는 4단계 회원가입

현재 회원 대시보드 스냅샷은 같은 `outputs` 폴더의 `ktalk-ai-member-dashboard`에 별도로 보관되어 있습니다.
